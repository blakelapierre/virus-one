About Virus One

Virus One is a plant cultivation automation, monitoring and reporting system designed by Rob Colbert as part of his medical and recreational cannabis cultivation activities. This software has been created as a labor of love after wasting over 20 years pursuing lies in Corporate America. Virus One is released to the cannabis cultivation community as an MIT-licensed open source contribution in the hope it will be used to improve the quality of cannabis being made available everywhere.

Virus One consists of many parts. There are temperature, humidity, light and other sensors connected to customized embedded systems. Devices are being read up to 1,000 times per second, and the information recorded in being processed, aggregated, sifted, filtered and sorted in a variety of ways to make reporting on and understanding the history of your grow a breeze. Each module reports to a centralized Node.js API server. The user interacts with an HTML5 web application on any device to observe and control their entire grow.

MongoDB

MongoDB is used as the database and backing store for the Virus One application. All information about all plants, strains, environments, modules, nutrients, etc., is stored in MongoDB. MongoDB allows Virus One to deliver on the promise of the "Internet of Things" (IoT) applied to Indoor Controlled Environment Agriculture.

MongoDB offers the Aggregation Framework, an application development framework that makes it very easy and intuitive to generate meaningful reports from all of the information collected during each grow cycle. It's never been easier to ask for average temperatures, growth rates and other interesting data.

Node.js

Node.js is used for pretty much everything in this entire application. The API server that allows alterations to the database is built using ExpressJS atop Node.js. The desktop application is hosted within node-webkit, a specialized project that integrates Node.js and a WebKit-based browser implementation highly specialized for desktop application development.

ExpressJS

ExpressJS is used to implement a very standard HTTP REST API for VirusOne. This allows the application to offer an enormous amount of functionality with minimal development effort.

Bunyan Logger

The Bunyan Logger is used to record information about events within the Virus One API server. This structured logging system helps others understand the inner workings of the server when watching it work from the command line in development environments. These structured logs are also immensely useful to Virus One's application developers when diagnosing problems.

HTTPS

Secure HTTP (HTTPS) is used for all communications between the Web application and the Virus One Node.js server. This helps to ensure the privacy of the data being communicated.

User Application

The Virus One user application (the part you're using right now) has been built as a standards-compliant HTML5 Web application. This decision allows the Virus One grow automation and monitoring software to be used and controlled on any device offering a standards-compliant HTML5 web browser. This includes most smart phones, tablets, laptop and desktop computers. Google Chrome and Mozilla Firefox are the recommended browsers for desktop computer use.

AngularJS

AngularJS is an HTML5 Web application development framework enabling rapid development of Rich Internet Application software. It offers standard ways to define services, resources, user actions, gestures and other features and application functionality in a way that helps manage large applications, keep them extensible and maintainable.

Semantic UI

Semantic is an HTML5 application development framework that helps developers create beautiful, responsive layouts using human-friendly HTML. It is similar to Twitter Bootstrap, but offers a different philosophy for Web develoment that is perhaps a bit more semantically correct, vastly more intuitive and requires a lot less HTML markup to implement amazing user interfaces.

Chart.js

All charting and graphing of data in Virus One is implemented using Chart.js. With six basic chart types and infinite control over appearance and user interactions, Virus One has the industry standard in charting and graphing with all features needed to provide enhanced data visualization and analysis for every grow cycle. You've never seen information about your grow presented like this.

QR-js

Virus One offers the ability to generate, print and scan QR codes per plant. Each plant receives a unique QR code that can be printed and used to easily and quickly identify each plant in your garden. Using a smartphone or tablet, the user merely points the device's camera at the QR code, Virus One loads that plant and the user can immediately interact with that plant's data.

Moment.js

Moment.js is a small library that implements a lot of features for manipulating and displaying date and time information. You are reading phrases such as "two weeks from now" and "six days ago" because Virus One uses Moment.js.
